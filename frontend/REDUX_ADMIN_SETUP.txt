HOW TO USE
================================================================================

1. IMPORT HOOKS IN COMPONENTS
   ```typescript
   import { useAppDispatch, useAppSelector } from '../../store';
   import { fetchBuyersAsync, setSearchQuery } from '../../store/adminBuyersSlice';
   ```

2. ACCESS STATE
   ```typescript
   const { buyers, loading, error } = useAppSelector(state => state.adminBuyers);
   ```

3. DISPATCH ACTIONS
   ```typescript
   const dispatch = useAppDispatch();
   dispatch(fetchBuyersAsync({ page: 1, pageSize: 10 }));
   ```

4. USE SELECTORS (RECOMMENDED)
   ```typescript
   import { selectBuyers, selectBuyersLoading } from '../../store/selectors/adminSelectors';
   const buyers = useAppSelector(selectBuyers);
   const loading = useAppSelector(selectBuyersLoading);
   ```

================================================================================
TROUBLESHOOTING
================================================================================

Issue: Data not loading
Solution: 
  1. Check backend is running (http://localhost:4000)
  2. Verify USE_MOCK_DATA = false in adminService.ts
  3. Check browser console for API errors
  4. Run window.testRedux.api() to diagnose

Issue: Search not working
Solution:
  1. Check debounce is working (500ms delay)
  2. Verify searchQuery is updated in Redux state
  3. Check API endpoint receives search parameter

Issue: Actions not updating UI
Solution:
  1. Check Redux DevTools for action dispatching
  2. Verify reducer is updating state correctly
  3. Check for any errors in console
